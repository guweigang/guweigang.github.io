恩，在Windows下用BAT调用PHP。


#+BEGIN_EXAMPLE
    TITLE PHP CLI BAT
COLOR B0
@echo off
cls
echo.
php -n test.php
echo.
pause

#+END_EXAMPLE


test.php内容：


#+BEGIN_EXAMPLE
    <?php
    echo "Hello, Wordl!\n";
    echo "你好，世界！";
