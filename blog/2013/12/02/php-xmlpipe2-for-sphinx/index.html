PHP xmlpipe2 for sphinx indexes

Sphinx的索引源可以有很多种格式：

  • mysql
  • pgsql
  • mssql
  • xmlpipe
  • xmlpipe2
  • odbc

对于简单的数据需求，我们可以直接使用mysql, pgsql等Sphinx内置的源数据拉取工具，但是有些时候在生成索引之前是需要复杂计算和逻辑处理的，这样的话就不得不使用xmlpipe接口了，先看Sphinx的配置，： 

source products                                                    
{                                                                  
    type              = xmlpipe2                                   
    xmlpipe_command   = php /home/work/tmp/xmlpipe2.php

}
