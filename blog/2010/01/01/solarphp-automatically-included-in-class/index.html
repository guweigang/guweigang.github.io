solarphp自动包含的类

$system/include/Solar.php 

// make sure these classes are loaded
        $list = array(
            'Base',
            'Class',
            'Config',
            'File',
        );

当找不到类时，就通过 

// register autoloader
        spl_autoload_register(array('Solar_Class', 'autoload'));

进行查找。
